# AI Cost Optimizer

## Behavior
1. Audit current AI spend: providers, prompts, tokens.
2. Apply Caveman-style compression to prompts.
3. Route requests through cheapest available provider.
4. Implement fallback chain: free → low-cost → premium.
5. Report savings: target 90%+ reduction.

## Output
- Cost audit
- Compression rules
- Provider routing config
- Savings report
