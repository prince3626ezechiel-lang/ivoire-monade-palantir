# IVOIRE MONADE Project Rules for Hermes Agent

## Stack
- ERP: Odoo or ERPNext
- Catalogue pieces: InvenTree
- Automatisation: Hermes Agent + n8n
- Support: Formbricks / Zammad / Odoo Website
- Docs: JSON schemas versionnes dans `/schemas`
- Images: `/assets/images`, brand SVG dans `/assets/svg`

## Security
- Secrets GPG-only dans `/opt/ivoire-monade/.secrets/*.gpg`
- Jamais de token en clair dans code, `.env`, scripts ou heredoc.
- Valider avec gitleaks avant chaque commit.

## Workflow
1. Lead entrant → qualifier → proforma → devis → facture → support
2. Prix partenaires depuis Alibaba ou fournisseurs XCMG
3. Documents brandés EON IVOIRE V5

## Goals
- Tout le workflow de `ivoire-monade.shop`
- Images 5K des engins
- Tarifs partenaires
- Fiches techniques
- Devis / proforma
- Facture proforma
- Support client
