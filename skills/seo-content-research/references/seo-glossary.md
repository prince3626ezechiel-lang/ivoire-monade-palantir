# SEO Glossary — Key Terms for Content Research

## Core SEO Concepts

| Term | Definition | Why It Matters |
|------|-----------|----------------|
| **Search Intent** | The goal behind a search query (Informational / Commercial / Transactional / Navigational) | Content must match intent — B2B buyer searching "supplier" wants a product page, not a blog post |
| **EEAT** | Experience, Expertise, Authoritativeness, Trustworthiness — Google's quality framework | Factory tours, certifications, case studies all signal EEAT |
| **SERP** | Search Engine Results Page | What ranks for a keyword determines competition level |
| **Long-tail Keyword** | Specific, 3-5 word phrases with lower search volume but higher conversion intent | B2B buyers use long-tail; easier to rank than head terms |
| **Head Term** | Short, broad keyword with high volume and high competition | "Tarpaulin" vs "PVC tarpaulin manufacturer Canada" |
| **TF-IDF / Semantic SEO** | Using related terms to signal topical relevance to search engines | Include synonyms and related concepts naturally in content |
| **CTR** | Click-Through Rate — percentage of searchers who click your result | Title + meta description optimization |
| **Bounce Rate** | Percentage of visitors who leave without interacting | High bounce = poor intent match or bad UX |
| **Dwell Time** | How long a visitor stays on your page | >2 min dwell = strong positive signal |

## Keyword Research Terms

| Term | Definition |
|------|-----------|
| **Seed Keyword** | The core term you start with (e.g., "tarpaulin") |
| **Keyword Difficulty (KD)** | How hard it is to rank for a term (0-100) |
| **Search Volume** | Monthly searches for a keyword |
| **LSI Keywords** | Semantically related terms (Google uses these to judge content depth) |
| **Fresh Keyword** | A keyword with growing search volume and low competition |

## International SEO Terms

| Term | Definition |
|------|-----------|
| **hreflang** | HTML tag telling Google which language a page targets |
| **ccTLD** | Country-code top-level domain (.ca, .ae, .de) |
| **gTLD** | Generic top-level domain (.com, .net) |
| **Geo-targeting** | Signals to Google which country a site targets |
| **Local Search** | Google results specific to a geographic location |

## Content Optimization

| Term | Definition |
|------|-----------|
| **Meta Title** | The clickable headline shown in SERP (50-60 chars) |
| **Meta Description** | The snippet below title (150-160 chars) |
| **H1 / H2 / H3** | Heading hierarchy — H1 = page title, H2 = main sections, H3 = subsections |
| **Internal Linking** | Links between pages on your site — spreads link equity |
| **External/Backlink** | Links from other sites to yours — primary ranking factor |
| **Anchor Text** | The clickable text of a link |
| **Cornerstone Content** | The most important piece of content on a topic — comprehensive, linked-to from all related pages |
| **Content Silos** | Grouping related content together to signal topical authority |
| **Thin Content** | Pages with little substance — penalized by Google |
| **Content Gap** | Keywords your competitors rank for but you don't |

## Technical SEO

| Term | Definition |
|------|-----------|
| **Core Web Vitals** | Google's page experience metrics (LCP, FID, CLS) |
| **Canonical URL** | The preferred version of a page (prevents duplicate content issues) |
| **Sitemap** | XML file listing all pages on a site |
| **Robots.txt** | Instructions for search engine crawlers |
| **Schema Markup** | Structured data that helps Google understand page content |

## B2B SEO Specific

| Term | Definition |
|------|-----------|
| **Alibaba Effect** | Chinese B2B platform dominating SERPs for manufacturer keywords |
| **Supplier Page** | Landing page optimized for procurement searches |
| **OEM/ODM Content** | Content targeting custom manufacturing buyers |
| **FOB/CIF Pricing** | International trade terms — including them signals B2B expertise |
| **Import Guide** | Article about importing from China — top-of-funnel B2B content |
