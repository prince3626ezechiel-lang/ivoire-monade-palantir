# BrowserOS External System Bridge

Light bridge skill sourced from `browseros-ai/skills` that enables Hermes to interface with the [BrowserOS](https://browseros.com) CLI for real browser automation, screenshots, content extraction, and agentic browsing.

## Source repo
- `https://github.com/browseros-ai/skills`
- Use the upstream skill name: `browseros`
- Preferred install / update path: upstream npm skill pack `browseros-ai/skills`

## Usage notes
- This skill is intentionally minimal because the canonical behavior lives in BrowserOS CLI.
- Invoke through packaging / CLI wrapper rather than duplicating command documentation here.
