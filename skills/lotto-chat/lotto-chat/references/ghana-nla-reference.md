# Ghana NLA Reference

## Games
- Lucky Tuesday / Thursday
- Fortune Thursday
- Friday Bonanza

## Format types
- Bankers : 1-3 numéros à jouer systématiquement
- 2-Sure : paire jugée quasi certaine
- Machine numbers : 5 numéros suggérés
- Stand-by / Perm : permutation en cas de doute
- Pyramide : séquence hiérarchique de chiffres

## Usage for CI
- Signaler les terminaisons dominantes GH vs CI
- Détecter paires récurrentes transfrontalières
- Importer méthode pyramidale/banker comme signal auxiliaire
- Ne JAMAIS transposer directement les tirages GH vers CI
