# CI Results Format

## Standard post structure
```
[Header]
Matrice de semaine côte d'Ivoire du JJ au JJ mois AAAA

[Catégorie]
Nom catégorie N°XXX 🇨🇮
Num1, Num2, Num3, Num4
Code: XXX
Event: XX
```

## Catégories fréquentes
- Lundi étoile / Mardi émergence / Mercredi fortune / Jeudi ...
- Réveil / Matinale / Akwaba / Sika
- Clavier / Afterwork / Vendredi solution
- Kado / Monni / Samedi Diamant / Soutra / Première heure

## Notes d'extraction
- Un post Instagram peut contenir 2-6 catégories
- Codes et Events ne sont pas toujours présents
- Numéros sont toujours 3 chiffres dans les screens observés
- La déduplication se fait par `category + numbers`
